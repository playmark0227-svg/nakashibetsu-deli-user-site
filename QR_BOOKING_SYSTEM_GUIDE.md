# QRコード予約システム - セットアップガイド

## 概要

ヘブンネット経由のお客様向けに、QRコードから簡単に予約できるシステムです。
高齢者の方でも使いやすいよう、大きな文字とシンプルなデザインで構築されています。

## システムフロー

```
1. ヘブンネット → お客様来店
2. 名刺配布（QRコード付き）
3. QRコード読み取り
4. 店舗のキャスト一覧表示
5. キャスト選択
6. 予約フォーム入力
7. 予約完了 → 店舗から確認電話
```

## QRコードの生成方法

### オンラインツールを使用（推奨）

1. **QRコード生成サイトにアクセス**
   - https://www.qr-code-generator.com/
   - https://www.the-qrcode-generator.com/
   - https://qr.quel.jp/ (日本語)

2. **URLを入力**
   - 上記の各店舗URLをコピー＆ペースト

3. **サイズとデザインを調整**
   - 推奨サイズ: 3cm × 3cm 以上
   - エラー訂正レベル: M または Q（中〜高）
   - 色: 黒（背景は白）
   - オプション: 店舗ロゴを中央に配置可能

4. **ダウンロード**
   - 形式: PNG または SVG（印刷用）
   - 解像度: 300dpi 以上（印刷品質）

### コマンドラインで生成（開発者向け）

```bash
# qrencode をインストール
sudo apt-get install qrencode  # Ubuntu/Debian
brew install qrencode          # macOS

# QRコード生成
qrencode -o zero-qr.png -s 10 "https://3002-il4xaruhtlkhd0yoxpuv3-c81df28e.sandbox.novita.ai/qr/11111111-1111-1111-1111-111111111111"
```

### デザインのポイント

1. **QRコードは大きく**
   - 最低 3cm × 3cm
   - 高齢者でもスキャンしやすいサイズ

2. **説明文はシンプルに**
   - 「スマホで読み取ると キャスト一覧が見れます」
   - 「簡単予約」などの分かりやすい言葉

3. **電話番号も明記**
   - QRコードが使えない方のため
   - 大きめのフォントで表示

4. **色使い**
   - QRコード: 黒
   - 背景: 白（コントラストが重要）
   - アクセント: 店舗のブランドカラー

## 高齢者向けデザインの特徴

### フォントサイズ

| 要素 | PCサイズ | モバイルサイズ |
|------|---------|---------------|
| ページタイトル | 60px | 48px |
| セクション見出し | 40px | 32px |
| 本文 | 24px | 20px |
| ボタンテキスト | 32px | 24px |
| 電話番号 | 48px | 36px |

### カラースキーム

- **メインカラー:** ローズ/ピンク（親しみやすい）
- **アクセント:** グリーン（出勤中）、グレー（休み）
- **背景:** ホワイト/ライトグレー（見やすい）
- **テキスト:** ダークグレー（コントラスト確保）

### UI要素

1. **大きなボタン**
   - 最小サイズ: 縦80px × 横100%
   - タップしやすい間隔: 16px以上

2. **明確なフィードバック**
   - ホバー時: スケール105%
   - タップ時: 色の変化
   - エラー時: 赤枠 + 絵文字

3. **視覚的な補助**
   - アイコンを多用
   - ステップ表示（1/3 → 2/3 → 3/3）
   - 進捗バー

4. **シンプルな入力**
   - 必須項目は最小限
   - プレースホルダーで例を表示
   - 入力形式の説明

## ページ構造

### 1. QRランディングページ (`/qr/[shopId]`)

**表示内容:**
- 店舗名（大きく）
- 店舗説明
- 電話番号（タップで発信）
- 営業時間
- キャスト一覧（出勤中を優先表示）

**特徴:**
- 出勤中のキャストは緑の背景で目立つ
- 各キャストカードは大きく表示
- タップで予約ページへ

### 2. 予約フォーム (`/qr/[shopId]/book/[girlId]`)

**入力項目:**
1. お名前（必須）
2. 電話番号（必須）
3. 希望日（必須）
4. 希望時間（必須）
5. コース（60分/90分/120分）
6. ご要望（任意）

**バリデーション:**
- リアルタイムエラー表示
- 送信前の確認
- エラー箇所へ自動スクロール

### 3. 完了画面

**表示内容:**
- 大きなチェックマーク
- 「ご予約ありがとうございます」
- 予約内容のサマリー
- 「お店から確認のお電話を差し上げます」
- 5秒後に自動でキャスト一覧へ戻る

## 名刺配布のタイミング

### 推奨配布タイミング

1. **初回来店時**
   - 受付時に配布
   - 「次回から簡単に予約できます」と説明

2. **会計時**
   - お見送りの際に配布
   - 「またのご利用お待ちしております」

3. **定期的な配布**
   - 紛失される方もいるため
   - 希望者にはいつでも配布

### 説明例

**シンプルな説明:**
```
「この名刺のQRコードを、スマホのカメラで
読み取るだけで、簡単に予約できますよ。

使い方が分からない場合は、いつでも
お電話くださいね。」
```

**詳しい説明（希望者向け）:**
```
1. スマホのカメラアプリを開く
2. QRコードにカメラを向ける
3. 画面に表示されるURLをタップ
4. キャスト一覧が表示されます
5. お好きなキャストを選んで予約！
```

## トラブルシューティング

### QRコードが読み取れない場合

**お客様への対応:**
1. 「お電話でご予約いただけます」
2. 電話番号を案内
3. 必要に応じて予約を代行

**技術的な原因:**
- QRコードの印刷品質が低い
- 照明が暗い/明るすぎる
- カメラのピントが合っていない
- QRコードが汚れている・折れている

**解決策:**
- 高解像度で印刷（300dpi以上）
- ラミネート加工
- 複数の名刺を用意

### ページが表示されない

**確認事項:**
1. インターネット接続
2. URLの正確性
3. サーバーの稼働状況

**お客様への対応:**
- 電話予約への誘導
- 別の方法（LINE等）の提案
