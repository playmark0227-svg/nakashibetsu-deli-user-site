# 中標津デリヘル情報サイト

ヘブンネット風のデリヘル情報サイトの叩き台です。中標津エリアの3店舗を想定した設計になっています。

## 🌟 主な機能

### ✅ 実装済み機能

#### 1. 年齢確認システム
- 18歳以上の確認ページ
- localStorage を使用した確認状態の保持
- 未確認ユーザーの自動リダイレクト

#### 2. トップページ
- **店舗選択セクション**: 3店舗の情報表示とリンク
- **新人紹介**: 最新の女の子4名を表示
- **人気ランキング**: アクセス数ベースのランキング（風営法対応）
- **店舗フィルター**: ランキングを店舗ごとに絞り込み

#### 3. 女の子一覧ページ
- グリッド表示（レスポンシブ対応）
- **高度な検索・フィルター機能**:
  - 名前検索
  - 店舗選択
  - 年齢範囲
  - 身長範囲
  - バストサイズ範囲
  - 対応オプション（複数選択可）
  - 新人のみ表示
- リアルタイム検索結果表示

#### 4. 女の子詳細ページ
- プロフィール情報（年齢、身長、スリーサイズ、血液型）
- 写真ギャラリー（サムネイル表示）
- チャームポイント
- 対応可能オプション
- 料金システム
- 出勤スケジュール（日付・時間帯）
- 口コミ・レビュー表示
- 電話予約・WEB予約ボタン

#### 5. 予約システム
- 日付・時間選択
- コース（時間・料金）選択
- お客様情報入力（名前、電話番号、メール）
- バリデーション機能
- 予約確認画面
- **✨ NEW: 予約確認メール送信**（お客様 + 店舗）
  - HTML形式の美しいメール
  - 予約内容の詳細
  - 店舗連絡先情報

#### 6. レビュー投稿機能
- 女の子選択
- 5段階評価
- コメント入力
- バリデーション機能
- **✨ NEW: レビュー通知メール送信**（管理者向け）
  - 新規口コミの即時通知
  - 承認・却下の判断依頼
- 5段階評価
- コメント入力
- バリデーション機能

#### 7. 店舗詳細ページ
- 店舗情報（電話、営業時間、住所）
- 料金システム
- 在籍キャスト一覧

#### 8. 管理画面（デモ版）
- キャスト一覧表示
- 検索・フィルター機能
- 統計情報表示
- 編集・削除ボタン（デモ版）

#### 9. レスポンシブデザイン
- モバイル、タブレット、デスクトップ対応
- ハンバーガーメニュー
- タッチフレンドリーなUI

### 📧 メール通知システム（NEW!）

**Resend API** を使用した本格的なメール配信機能を実装しました！

#### 予約確認メール
- **お客様向け**: 予約内容、店舗連絡先、注意事項
- **店舗向け**: 新規予約通知、お客様情報、確認依頼
- HTML形式の美しいデザイン
- レスポンシブ対応

#### レビュー通知メール
- **管理者向け**: 新規口コミ投稿の即時通知
- 承認・却下の判断依頼
- 口コミ内容のプレビュー

#### セットアップ方法
詳細は [SETUP_EMAIL.md](./SETUP_EMAIL.md) を参照してください。

```bash
# 環境変数を設定
cp .env.example .env.local

# .env.local を編集してResend APIキーを設定
RESEND_API_KEY=your_api_key_here
```

**無料プラン**: 月間3,000通まで無料で使用可能！

## 🚀 セットアップ方法

### 必要な環境
- Node.js 18.x 以上
- npm または yarn

### インストール

```bash
# リポジトリのクローン
cd nakashibetsu-deli

# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:3000` を開いてアクセスできます。

## ⚖️ 風営法対応について

このサイトは以下の点で風営法に配慮しています：

1. **年齢確認**: 18歳未満のアクセス制限
2. **ランキング表示**: アクセス数ベースの客観的指標を使用
3. **情報提供**: あくまで情報サイトとして設計
4. **口コミ管理**: 管理者承認制を想定

**注意**: 実際の運用時は、必ず弁護士に相談し、最新の風営法規制を確認してください。
