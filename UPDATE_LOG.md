## 📧 実装されたメール機能

### 1. 予約確認メール（お客様向け）

予約が完了すると、お客様に自動で確認メールが送信されます。

**含まれる内容：**
- ✅ 予約内容の詳細（女の子、日時、コース、料金）
- ✅ 店舗の連絡先情報
- ✅ 重要な注意事項
- ✅ 美しいHTMLデザイン（グラデーション、レスポンシブ対応）

**メールサンプル：**
```
━━━━━━━━━━━━━━━━━━━
予約受付完了
ご予約ありがとうございます
━━━━━━━━━━━━━━━━━━━

〇〇 様

この度はプレミアム中標津をご利用いただき、
誠にありがとうございます。
以下の内容でご予約を承りました。

【ご予約内容】
女の子: あやか
日時: 2026年1月20日 18:00〜
コース: 90分コース
料金: ¥21,000

⚠️ 重要なお知らせ
・店舗から確認のお電話をさせていただきます
・確認が取れ次第、ご予約が正式に確定となります
```

### 2. 予約通知メール（店舗向け）

新規予約が入ると、店舗に即座に通知メールが送信されます。

**含まれる内容：**
- ✅ 新規予約の通知
- ✅ お客様情報（名前、電話番号、メール）
- ✅ 予約詳細（指名、日時、コース）
- ✅ 次のアクション（確認電話の依頼）

### 3. レビュー通知メール（管理者向け）

口コミが投稿されると、管理者に即座に通知メールが送信されます。

**含まれる内容：**
- ✅ 新規口コミの内容
- ✅ 評価（星の数）
- ✅ 投稿者情報
- ✅ 承認・却下の判断依頼
- ✅ 管理画面へのリンク

## 🔧 技術的な実装

### API Routes

#### `/api/bookings` - 予約API
- POST リクエストで予約データを受信
- バリデーション処理
- お客様と店舗へメール送信
- レスポンス返却

#### `/api/reviews` - レビューAPI
- POST リクエストでレビューデータを受信
- バリデーション処理
- 管理者へメール送信
- レスポンス返却

### メールテンプレート

`lib/email/templates.ts` に3種類のHTMLメールテンプレートを実装：

1. `sendBookingConfirmationEmail()` - お客様向け予約確認
2. `sendBookingNotificationToShop()` - 店舗向け予約通知
3. `sendReviewNotificationEmail()` - 管理者向けレビュー通知

**特徴：**
- 完全なHTMLメール（インラインCSS）
- グラデーション背景
- レスポンシブデザイン
- 日本語完全対応
- 絵文字使用

## 🚀 セットアップ方法

### ステップ1: Resendアカウント作成

1. https://resend.com にアクセス
2. 無料アカウントを作成
3. APIキーを取得（月3,000通まで無料！）

### ステップ2: 環境変数の設定

`.env.local` ファイルを編集：

```bash
# Resend API Key
RESEND_API_KEY=re_あなたのAPIキー

# メール送信元（独自ドメイン設定後）
EMAIL_FROM=noreply@nakashibetsu-deli.com

# 店舗通知用メールアドレス
SHOP_EMAIL_PREMIUM=premium@example.com
SHOP_EMAIL_ROYAL=royal@example.com
SHOP_EMAIL_ELEGANCE=elegance@example.com

# 管理者メールアドレス
ADMIN_EMAIL=admin@example.com
```

### ステップ3: サーバー再起動

```bash
npm run dev
```

### ステップ4: テスト送信

1. 予約フォームで自分のメールアドレスを入力
2. 予約を実行
3. メールが届くか確認！

## 💰 料金について

### Resend 料金プラン

| プラン | 月額 | 送信数/月 | 推奨用途 |
|--------|------|-----------|----------|
| **Free** | **¥0** | **3,000通** | 開発・小規模運用 |
| Pro | $20 | 50,000通 | 中規模ビジネス |
| Business | カスタム | カスタム | 大規模ビジネス |

**無料プランで十分な理由：**
- 1日100通 × 30日 = 3,000通
- 予約1件 = 2通（お客様 + 店舗）
- レビュー1件 = 1通（管理者）
- **月間1,000件の予約に対応可能！**
